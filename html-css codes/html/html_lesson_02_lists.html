Lists In HTML There are two main kinds of lists - ordered and unordered.

Ordered lists

Use this for a numbered list as follows
  1. Get up
  2. Get ready
  3. Get to office
  4. Come back
  5. Sleep

Unordered lists

Use this for a simple bulleted list
  • Call the plumber
  • Buy Groceries
  • Pay the bills

Nested lists

You can nest lists as well.
  1. Pick up groceries
    • Sugar
    • Salt
    • Rice
  2. Pay the bills
  3. Sell the car

Styling lists

You can style lists in interesting ways with css, that is covered in the css unit.
